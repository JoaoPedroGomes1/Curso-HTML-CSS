Link:css

É usado para criar um link ppara uma página especifica de CSS, onde vai ficar os estilos do seu site, usado em baixo do tittle em head.

E também pode ser usado 2 link com arquivos CSS em uma página, ele adiciona o conteudo dos dois arquivos, e também posso usar os outros tipos de CSS também.

Existe uma ierarquia nos estilos no qual o mais poderoso permanece, sendo que os outros não podem mudalo, primeiro o Enline, depois o local e por último o 
externo, o enline é o que vc vai na tag e coloca, o local é aquele que se usa no head em baixo do tittle e o externo é o utilizado por link.

Observações:

 evitar usar o css enline, só em casos pontuais que só se vai utilizar em 1 página

 css interno dentro do head é bom utilizar em estilos pequenos, como 1 a 3 páginas para não dificultar o manejo dos estilos

 css externos, usado pelo link, é sempre bom ser usado e sem moderação kkk, uma boa escolha para sites com muitas páginas ou no mínimo 4 ou até menos.

position: relative;

Usado para deixar o container (box) em posicionamento relativo, no padrão ele já vem assim

Ex:  #container {
            background-image: url('imagens/target001.png');
            background-size: 100% 100%;

            position: relative;
            height: 96vh;
            background-color: purple;
            padding: 10px;
        }

( no curso, fala que o que está de fora, está contendo (container) tem o posicionamento relative, e o conteúdo que está dentro tem posicionamento absolute )


===================================================================================================================================================================================================================================

position: absolute;

Usado para deixar o conteúdo com posicionamento absolute

Podendo ajustar ou centralizar o conteúdo, usando left e top tanto com px que pode ir ajustando, ou % que deve ser feito outros ajustes

Ex:  left: 100px;		 left: 50%;	
       top: 100px;		top: 50%		

no caso se usar %, pode acontecer que não fique centralizado a imagem, ele considera a ponta esquerda superior, não o cecntro do conteúdo, nesse caso se utiliza

transform: translate(-50%, -50%);

que pode ser ajustado no px ou % também, melhor usar % como no exemplo a baixo

Ex: #conteudo {
            background-image: url('imagens/target001.png');
            background-size: 100% 100%;

            position: absolute;
            height: 200px;
            width: 400px;
            background-color: yellow;
            left: 50%;
            top: 50%;
            transform: translate(-50%, -50%);
        }

( no curso, fala que o que está de fora, está contendo (container) tem o posicionamento relative, e o conteúdo que está dentro tem posicionamento absolute )

===================================================================================================================================================================================================================================

transform: translate(-50%, -50%);

Usado para ajustar ou centralizar o conteúdo, nos exemplos usados foram utilizados valores negativos, mas apenas para se adequar a imagem utilizada, pode ser necessario usar tanto positivo quanto negativo, depende do caso
